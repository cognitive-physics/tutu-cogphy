# 四维责任模型

> 一个事后归因分解器：把已发生事件拆成四个正交因果分量，再由第四维导出行动路径。

## 模型概述

四维责任模型不是做价值审判，而是做**能量审计**：

- 把因果拆开
- 找到真正的杠杆点
- 尽量减少无效耗散

## 四个维度

### 1. 行为责任 · 观测量

客观锁定已发生的事件与言行：

```math
A_1 = ρ(t_n)
```

只记录“发生了什么”，不带解释。

### 2. 教育责任 · 初始条件回溯

回溯导致行为的潜在模式与根源：

```math
A_2 = f(ρ_0, H_0, λ_{history})
```

对应先天结构、早期动力学以及成长环境中的耦合历史。

### 3. 环境责任 · 边界条件分离

分离外界输入造成的偏移：

```math
A_3 = Δρ_{ext} = ρ(t_n) - ρ_{自生}(t_n)
```

作用是把“我的问题”和“环境的问题”拆开。

### 4. 成长责任 · `δA = 0`

在前三维信息已知时，选择使认知耗散最小的行动路径：

```math
δA = 0
\\quad \\Rightarrow \\quad
\\text{在所有可能路径中选择认知耗散最小的一条}
```

它不问“什么是对的”，只问：

- 哪条路径闭合最快
- 哪条路径耗散最少
- 哪条路径最接近稳态吸引子 `σ*`

## 运作流程

1. 锁定当前事件
2. 回溯初始条件与耦合历史
3. 分离环境边界条件
4. 导出最小耗散行动策略

## 为什么它有效

1. **正交分解**：四维互不重叠，覆盖较完整
2. **不做道德审判**：只做效率和能量分析
3. **第四维导出行动**：分析不会停在空转层
4. **天然防内耗**：因果拆清后，不会把所有压力都压回自己身上
